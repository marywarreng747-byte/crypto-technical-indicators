# crypto-technical-indicators
# Crypto Technical Indicators

Python script to fetch crypto data, calculate technical indicators (RSI, MACD, Bollinger Bands, etc.), generate signals, and plot them interactively.

## Features
- Uses `ccxt` for exchange data (Binance by default)
- `pandas_ta` for 100+ indicators
- Interactive Plotly charts
- Simple buy/sell signal logic (customizable)

## Installation

```bash
pip install -r requirements.txt
